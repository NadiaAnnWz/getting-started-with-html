Introduction
-The web is used to access information by way of computers.
-The user types a URL into their browser, or searches for one, which results in the computer communicating with servers by way of HTTP and the Internet.
-The accessed information is in the form of webpages that contain HTML (hypertext markup language).

Aspects of HTML
-Text: visible
-Markup: how it is arranged
-References to other documents: images and videos
-Links to other pages.

Tags
-What markup is made of: opening and closing tags
-Example (no spaces) < tag name > content < / tag name >
-A basic tag would be one to make text bold: (no spaces) < b > text < / b >

Whitespace and Line Breaks
-To create multiple lines, the tags used are for Breaks (br) and Paragraph (p)
-Breaks are considered a 'Void Tag' meaning they do not have a closing tag.
-Paragraphs require a closing tag. Paragraph tags essentially create a 'box' in which the text is written, of which we will eventually learn to format the size.

Inline vs. Block Elements
-Inline elements results in changes to a line of text. Tags include attribute, span, break, image, and strong.
-Block elements format entire blocks of information. They include div, form, and paragraph.
